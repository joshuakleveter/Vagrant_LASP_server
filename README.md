# Vagrant_LASP_server
Vagrantfile, shell provisioner, Apache conf file, and directory structure for a basic LASP (Linux, Apache, SQLite, PHP) server.

## System Requirements
To use this Vagrantfile you will need a basic Vagrant setup.  This includes [Vagrant](https://www.vagrantup.com/) and a VM provider, such as Oracle's [VirtualBox](https://www.virtualbox.org).  For more information on how to use Vagrant, see the Vagrant *[Getting Started](https://docs.vagrantup.com/v2/getting-started/index.html)* page.
